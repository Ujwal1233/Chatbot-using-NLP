Installation and Setup
Pre-requisites
You need to have Python 3.x installed on your system.
An OpenAI API key Obtain it from the OpenAI website.
Steps to Run the Chatbot
Run the following command at command prompt/terminal/console: git clone https://github.com/your-username/your-repo-name.git 
Use pip install -r requirements.txt to install the required dependencies.
Replace the OPENAI_API_KEY variable in the code with your actual OpenAI API key 
Finally run the chatbot with python app.py (app.py being your application's main python file).

Usage
Communicate with the chatbot entering your messages within the input field and pressing the "Send" button.
Queries would be answered as per its NLP capabilities and OpenAI integration on the chatbot.
